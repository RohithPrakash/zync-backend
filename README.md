# Zync

Zync, a chat application build to understand real-time data transfer.

[Demo](https://modest-pare-a8eeab.netlify.app/ "Zync's Homepage")

## How To Run

* Clone the repository
* Install dependencies using npm or yarn
  ```
  npm install
  ```
  or
  ```
  yarn install
  ```
* To host locally
  ```
  npm start
  ```
